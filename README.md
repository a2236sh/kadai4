##### kadai4
##### システムソフトウェア特論 課題4 : 開発

##### 提出物
- .travis.yml
  - TravisCI設定ファイル
- gcd.sh
  - 最大公約数を計算するスクリプトファイル
- gcd_test.sh
  - gcd.sh の挙動を確認するテストを記したファイル
  - 期待する結果とスクリプトの結果が同値であるかを確認する関数を定義し、
  正常系と異常系に分けてテストを実行しています。テストに失敗するとCIがコケるようになっています
